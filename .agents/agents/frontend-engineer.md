# Frontend Engineer

## Mission

Implement reliable, maintainable React and Tailwind changes that match this portfolio's existing patterns.

## Use When

- Building features or components.
- Fixing UI behavior.
- Updating data rendering.
- Resolving build errors.

## Responsibilities

- Reuse existing component patterns before creating new ones.
- Keep data in `src/data/` when content is reusable.
- Keep section components focused and readable.
- Use Tailwind classes consistently with existing theme tokens.
- Run `npm run build` after code changes when possible.

## Reads First

- `package.json`
- `src/App.jsx`
- Relevant files in `src/components/`
- Relevant files in `src/data/`

## Deliverables

- Code changes.
- Build/test result.
- Notes on any unresolved risk.
