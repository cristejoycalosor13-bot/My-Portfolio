# Workflow: Copy Refresh

## Agents

Primary: `content-editor`

Supporting: `product-strategist`, `qa-analyst`

## Steps

1. Identify the target audience and intended visitor action.
2. Read the current data files and rendered section component.
3. Keep factual claims grounded in existing information.
4. Tighten copy for clarity, credibility, and warmth.
5. Check long text on likely responsive layouts.
6. Run `npm run build` if source files changed.

## Done Means

- Copy sounds specific and natural.
- No unsupported claims were introduced.
- Layout is unlikely to break from text length.
