# My Portfolio

Personal portfolio built with Vite, React, and Tailwind CSS.

## Agent Workflow

This repo includes a durable multi-agent workflow in `.agents/`.

Start here:

- `AGENTS.md`
- `.agents/README.md`
- `.agents/registry.md`

Use the workflow whenever making feature, content, design, QA, release, or maintenance changes.

## Development

```bash
npm install
npm run dev
npm run build
```
